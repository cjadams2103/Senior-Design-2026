The Thingspeak and Nano Esp32 PDF is only for showing how to setup Thingspeak and downloading the Arduino IDE. DO NOT USE THE CODE AT THE BOTTOM. Use the code provided in the SeniorDesign26.zip

On the Arduino IDE, the Board Managers to install are:
  Arduino ESP32 Boards by Arduino, Version 2.0.18-arduino.5

On the Arduino IDE, the Libraries to install are:
  ThingSpeak by MathWorks, Version 2.1.1
  Arduino_ESP32_OTA by Arduino, Version 0.3.1
  Adafruit_BusIO by Adafruit, Version 1.17.4
  Adafruit GFX Library by Adafruit, Version 1.12.5
  Adafruit SSD1306 by Adafruit, Version 2.5.16
  LiquidCrystal_I2C by Martin Kubovclk, Version 2.0.0
  PS2Keyboard by Christian Weichel, Version 2.5

Not all of these libraries are required for the basic operation in SeniorDesign26.ino but in case of additional peripherals, it is best to add them all anyways.
